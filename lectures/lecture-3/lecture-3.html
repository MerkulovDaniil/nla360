–í–∞–∂–Ω–æ–π –∑–∞–¥–∞—á–µ–π –≤–æ –º–Ω–æ–≥–∏—Ö –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è—Ö —è–≤–ª—è–µ—Ç—Å—è –Ω–∞—Ö–æ–∂–¥–µ–Ω–∏–µ –∞–ø–ø—Ä–æ–∫—Å–∏–º–∞—Ü–∏–∏ –Ω–∏–∑–∫–æ–≥–æ —Ä–∞–Ω–≥–∞ –¥–ª—è –∑–∞–¥–∞–Ω–Ω–æ–π –º–∞—Ç—Ä–∏—Ü—ã —Å –∑–∞–¥–∞–Ω–Ω–æ–π —Ç–æ—á–Ω–æ—Å—Ç—å—é \epsilon –∏–ª–∏ —Ä–∞–Ω–≥–æ–º r.
–ü—Ä–∏–º–µ—Ä—ã: * –∞–Ω–∞–ª–∏–∑ –≥–ª–∞–≤–Ω—ã—Ö –∫–æ–º–ø–æ–Ω–µ–Ω—Ç * —Ä–µ–∫–æ–º–µ–Ω–¥–∞—Ç–µ–ª—å–Ω—ã–µ —Å–∏—Å—Ç–µ–º—ã * –º–µ—Ç–æ–¥ –Ω–∞–∏–º–µ–Ω—å—à–∏—Ö –∫–≤–∞–¥—Ä–∞—Ç–æ–≤ * —Å–∂–∞—Ç–∏–µ –Ω–µ–π—Ä–æ–Ω–Ω—ã—Ö —Å–µ—Ç–µ–π

–≠—Ç–∏ –∑–∞–¥–∞—á–∏ –º–æ–≥—É—Ç –±—ã—Ç—å —Ä–µ—à–µ–Ω—ã —Å –ø–æ–º–æ—â—å—é SVD.

–°–∏–Ω–≥—É–ª—è—Ä–Ω–æ–µ —Ä–∞–∑–ª–æ–∂–µ–Ω–∏–µ

–î–ª—è –≤—ã—á–∏—Å–ª–µ–Ω–∏—è –∞–ø–ø—Ä–æ–∫—Å–∏–º–∞—Ü–∏–∏ –Ω–∏–∑–∫–æ–≥–æ —Ä–∞–Ω–≥–∞ –Ω–∞–º –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –≤—ã—á–∏—Å–ª–∏—Ç—å —Å–∏–Ω–≥—É–ª—è—Ä–Ω–æ–µ —Ä–∞–∑–ª–æ–∂–µ–Ω–∏–µ (SVD).

–¢–µ–æ—Ä–µ–º–∞ –õ—é–±–∞—è –º–∞—Ç—Ä–∏—Ü–∞ A\in \mathbb{C}^{n\times m} –º–æ–∂–µ—Ç –±—ã—Ç—å –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ –∫–∞–∫ –ø—Ä–æ–∏–∑–≤–µ–¥–µ–Ω–∏–µ —Ç—Ä–µ—Ö –º–∞—Ç—Ä–∏—Ü:

A = U \Sigma V^*,

–≥–¥–µ - U - —É–Ω–∏—Ç–∞—Ä–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞ —Ä–∞–∑–º–µ—Ä–∞ n \times K, - V - —É–Ω–∏—Ç–∞—Ä–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞ —Ä–∞–∑–º–µ—Ä–∞ m \times K, K = \min(m, n), - \Sigma - –¥–∏–∞–≥–æ–Ω–∞–ª—å–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞ —Å –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–º–∏ —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ \sigma_1 \geq \ldots, \geq \sigma_K –Ω–∞ –¥–∏–∞–≥–æ–Ω–∞–ª–∏. - –ë–æ–ª–µ–µ —Ç–æ–≥–æ, –µ—Å–ª–∏ \text{rank}(A) = r, —Ç–æ \sigma_{r+1} = \dots = \sigma_K = 0.

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ

  • –ú–∞—Ç—Ä–∏—Ü–∞ A^*A —è–≤–ª—è–µ—Ç—Å—è —ç—Ä–º–∏—Ç–æ–≤–æ–π, —Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, –¥–∏–∞–≥–æ–Ω–∞–ª–∏–∑–∏—Ä—É–µ–º–∞ –≤ —É–Ω–∏—Ç–∞—Ä–Ω–æ–º –±–∞–∑–∏—Å–µ (–±—É–¥–µ—Ç –æ–±—Å—É–∂–¥–∞—Ç—å—Å—è –¥–∞–ª–µ–µ –≤ –∫—É—Ä—Å–µ).
  • A^*A\geq0 (–Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∞), –ø–æ—ç—Ç–æ–º—É —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è –Ω–µ–æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã. –°–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, —Å—É—â–µ—Å—Ç–≤—É–µ—Ç —É–Ω–∏—Ç–∞—Ä–Ω–∞—è –º–∞—Ç—Ä–∏—Ü–∞ V = [v_1, \dots, v_n] —Ç–∞–∫–∞—è, —á—Ç–æ

V^* A^* A V = \text{diag}(\sigma_1^2,\dots, \sigma_n^2), \quad \sigma_1\geq \sigma_2\geq \dots \geq \sigma_n.

–ü—É—Å—Ç—å \sigma_i = 0 –¥–ª—è i>r, –≥–¥–µ r - –Ω–µ–∫–æ—Ç–æ—Ä–æ–µ —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ.
–ü—É—Å—Ç—å V_r= [v_1, \dots, v_r], \Sigma_r = \text{diag}(\sigma_1, \dots,\sigma_r). –°–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ

V^*_r A^* A V_r = \Sigma_r^2 \quad \Longrightarrow \quad (\Sigma_r^{-1} V_r^* A^*) (A V_r\Sigma_r^{-1} ) = I.

–í —Ä–µ–∑—É–ª—å—Ç–∞—Ç–µ, –º–∞—Ç—Ä–∏—Ü–∞ U_r = A V_r\Sigma_r^{-1} —É–¥–æ–≤–ª–µ—Ç–≤–æ—Ä—è–µ—Ç U_r^* U_r = I –∏, —Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, –∏–º–µ–µ—Ç –æ—Ä—Ç–æ–≥–æ–Ω–∞–ª—å–Ω—ã–µ —Å—Ç–æ–ª–±—Ü—ã.
–î–æ–±–∞–≤–∏–º –∫ U_r –ª—é–±—ã–µ –æ—Ä—Ç–æ–≥–æ–Ω–∞–ª—å–Ω—ã–µ —Å—Ç–æ–ª–±—Ü—ã, –∫–æ—Ç–æ—Ä—ã–µ –æ—Ä—Ç–æ–≥–æ–Ω–∞–ª—å–Ω—ã —Å—Ç–æ–ª–±—Ü–∞–º –≤ U_r, –∏ –æ–±–æ–∑–Ω–∞—á–∏–º —ç—Ç—É –º–∞—Ç—Ä–∏—Ü—É –∫–∞–∫ U. –¢–æ–≥–¥–∞

AV = U \begin{bmatrix} \Sigma_r & 0 \\ 0 & 0 \end{bmatrix}\quad \Longrightarrow \quad U^* A V = \begin{bmatrix}\Sigma_r & 0 \\ 0 & 0 \end{bmatrix}.

–ü–æ—Å–∫–æ–ª—å–∫—É —É–º–Ω–æ–∂–µ–Ω–∏–µ –Ω–∞ –Ω–µ–≤—ã—Ä–æ–∂–¥–µ–Ω–Ω—ã–µ –º–∞—Ç—Ä–∏—Ü—ã –Ω–µ –º–µ–Ω—è–µ—Ç —Ä–∞–Ω–≥ A, –º—ã –∏–º–µ–µ–º r = \text{rank}(A).

–°–ª–µ–¥—Å—Ç–≤–∏–µ 1: A = \displaystyle{\sum_{\alpha=1}^r} \sigma_\alpha u_\alpha v_\alpha^* –∏–ª–∏ –ø–æ—ç–ª–µ–º–µ–Ω—Ç–Ω–æ a_{ij} = \displaystyle{\sum_{\alpha=1}^r} \sigma_\alpha u_{i\alpha} \overline{v}_{j\alpha}

–°–ª–µ–¥—Å—Ç–≤–∏–µ 2: \text{ker}(A) = \mathcal{L}\{v_{r+1},\dots,v_n\}

\text{im}(A) = \mathcal{L}\{u_{1},\dots,u_r\}

\text{ker}(A^*) = \mathcal{L}\{u_{r+1},\dots,u_n\}

\text{im}(A^*) = \mathcal{L}\{v_{1},\dots,v_r\}

–¢–µ–æ—Ä–µ–º–∞ –≠–∫–∞—Ä—Ç–∞-–Ø–Ω–≥–∞

–ù–∞–∏–ª—É—á—à–µ–µ –ø—Ä–∏–±–ª–∏–∂–µ–Ω–∏–µ –Ω–∏–∑–∫–æ–≥–æ —Ä–∞–Ω–≥–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å –≤—ã—á–∏—Å–ª–µ–Ω–æ —Å –ø–æ–º–æ—â—å—é SVD.

–¢–µ–æ—Ä–µ–º–∞: –ü—É—Å—Ç—å r < \text{rank}(A), A_r = U_r \Sigma_r V_r^*. –¢–æ–≥–¥–∞

\min_{\text{rank}(B)=r} \|A - B\|_2 = \|A - A_r\|_2 = \sigma_{r+1}.

–¢–æ –∂–µ —Å–∞–º–æ–µ –≤–µ—Ä–Ω–æ –¥–ª—è \|\cdot\|_F, –Ω–æ \|A - A_r\|_F = \sqrt{\sigma_{r+1}^2 + \dots + \sigma_{\min (n,m)}^2}.

–î–æ–∫–∞–∑–∞—Ç–µ–ª—å—Å—Ç–≤–æ

  • –ü–æ—Å–∫–æ–ª—å–∫—É \text{rank} (B) = r, —Ç–æ \text{dim}~\text{ker}~B = n-r.
  • –°–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, —Å—É—â–µ—Å—Ç–≤—É–µ—Ç z\not=0 —Ç–∞–∫–æ–π, —á—Ç–æ z\in \text{ker}(B) \cap \mathcal{L}(v_1,\dots,v_{r+1}) (—Ç–∞–∫ –∫–∞–∫ \text{dim}\{v_1,\dots,v_{r+1}\} = r+1).
  • –ó–∞—Ñ–∏–∫—Å–∏—Ä—É–µ–º \|z\| = 1. –¢–æ–≥–¥–∞,

\|A-B\|_2^2 \geq \|(A-B)z\|_2^2 = \|Az\|_2^2 = \| U\Sigma V^* z\|^2_2= \|\Sigma V^* z\|^2_2 = \sum_{i=1}^{n} \sigma_i^2 (v_i^*z)^2 =\sum_{i=1}^{r+1} \sigma_i^2 (v_i^*z)^2 \geq \sigma_{r+1}^2\sum_{i=1}^{r+1} (v_i^*z)^2 = \sigma_{r+1}^2

—Ç–∞–∫ –∫–∞–∫ \sigma_1\geq \dots \geq \sigma_{r+1} –∏ \sum_{i=1}^{r+1} (v_i^*z)^2 = \|V^*z\|_2^2 = \|z\|_2^2 = 1.

–û—Å–Ω–æ–≤–Ω–æ–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç –æ –ø—Ä–∏–±–ª–∏–∂–µ–Ω–∏–∏ –Ω–∏–∑–∫–æ–≥–æ —Ä–∞–Ω–≥–∞

–°–ª–µ–¥—Å—Ç–≤–∏–µ: –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ –Ω–∞–∏–ª—É—á—à–µ–≥–æ –ø—Ä–∏–±–ª–∏–∂–µ–Ω–∏—è —Ä–∞–Ω–≥–∞ r —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω–æ —É—Å—Ç–∞–Ω–æ–≤–∫–µ \sigma_{r+1}= 0, \ldots, \sigma_K = 0. –û—à–∏–±–∫–∞

\min_{A_r} \Vert A - A_r \Vert_2 = \sigma_{r+1}, \quad \min_{A_r} \Vert A - A_r \Vert_F = \sqrt{\sigma_{r+1}^2 + \dots + \sigma_{K}^2}

–≤–æ—Ç –ø–æ—á–µ–º—É –≤–∞–∂–Ω–æ —Å–º–æ—Ç—Ä–µ—Ç—å –Ω–∞ —Å–∫–æ—Ä–æ—Å—Ç—å —É–±—ã–≤–∞–Ω–∏—è —Å–∏–Ω–≥—É–ª—è—Ä–Ω—ã—Ö –∑–Ω–∞—á–µ–Ω–∏–π.
